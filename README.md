# MAUI-.NET-Aspire
.NET Saturday Pordenone 2026

Conosciamo .NET Aspire per i servizi backend, ma sapevi che può essere molto utile anche in WPF e MAUI? In questa sessione scopriamo insieme come connettere la nostra app client alla dashboard di Aspire, sfruttando service discovery, telemetria e metriche per creare applicazioni più intelligenti e reattive.
